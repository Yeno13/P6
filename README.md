Sixth project from OpenClassroom: Construisez une API sécurisée pour une application d'avis gastronomiques

INSTALLATION

1) Dépendances à installer :

    NodeJS 14.0.+
    Angular CLI 7.0.2.+
    node-sass

2)Créer une base de donnée sur AtlasMongoDB. Dans le dossier backend, créer un fichier nommé .env dans ce fichier y renseigner: 

    TOKEN='votre_code_secret'
    MONGODB_USER='nom_utilisateur_mongodb'
    MONGODB_PASSWORD='mot_de_passe_mongodb'
    MONGODB_CLUSTER_NAME='nom_du_cluster_mongodb'

3) Utilisez les commandes npm install puis npm start à partir du dossier backend. Pour le fonctionnement du frontend veuillez vous référer au README fourni dans le dossier.
